Internet deve essere al servizio del "bene pubblico": il manifesto di Tim Berners-Lee

Scritto da The Booleaner

10/10/2020

Il web compie 32 anni e il suo creatore, Berners-Lee, ha scritto un lungo e articolato manifesto per sottolineare i punti di forza della rete, ma anche i suoi punti deboli. Il testo è stato pubblicato sul sito della Web Foundation, l'organizzazione no-profit fondata da Berners-Lee per promuovere l'accesso universale alla rete e la sua libertà d'uso.

  • Il web è un diritto umano
  • Il web deve essere accessibile a tutti
  • Il web deve essere aperto e libero
  • Il web deve essere sicuro e rispettoso della privacy
  • Il web deve essere trasparente e responsabile

"Il web è uno strumento potente che può aiutare a costruire un mondo migliore. Per farlo, dobbiamo garantire che il web serva i bisogni di tutti le persone, di tutte le comunità e di tutte le nazioni. Questo è il motivo per cui ho lanciato la Web Foundation. E questo è il motivo per cui lavoro per garantire che il web rimanga un bene pubblico che tutti possano utilizzare liberamente, per comunicare, collaborare e innovare."
